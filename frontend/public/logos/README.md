## Logo Specifications

- **Format**: PNG (preferred) or SVG
- **Background**: Transparent
- **Size**: 300x150px minimum (will be auto-scaled)
- **Quality**: High resolution for crisp display

## Where to Get Logos

You can download official logos from:
- Official brand websites
- Press kits from each platform
- Logo repositories (ensure you have rights to use them)

## Fallback

If a logo file is missing, the platform name will be displayed as text instead.
